Classes for parsing and processing {@link javax.xml.crypto.dsig.keyinfo.KeyInfo KeyInfo} elements and structures. KeyInfo is an optional element that enables the recipient(s) to obtain the key needed to validate an {@link javax.xml.crypto.dsig.XMLSignature XMLSignature}. KeyInfo may contain keys, names, certificates and other public key management information, such as in-band key distribution or key agreement data. This package contains classes representing types defined in the W3C specification for XML Signatures, such as {@link javax.xml.crypto.dsig.keyinfo.KeyName KeyName}, {@link javax.xml.crypto.dsig.keyinfo.KeyValue KeyValue}, {@link javax.xml.crypto.dsig.keyinfo.RetrievalMethod RetrievalMethod}, {@link javax.xml.crypto.dsig.keyinfo.X509Data X509Data}, {@link javax.xml.crypto.dsig.keyinfo.X509IssuerSerial X509IssuerSerial}, and {@link javax.xml.crypto.dsig.keyinfo.PGPData PGPData}. {@link javax.xml.crypto.dsig.keyinfo.KeyInfoFactory KeyInfoFactory} is an abstract factory that creates KeyInfo objects from scratch.
